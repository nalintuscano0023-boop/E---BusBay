SafarSathi 🚌

>Overview:

SafarSathi is a Smart Bus Tracking and Ticket Booking desktop application built using Python and Tkinter.

It is designed to help users track buses in real time, search routes, check ETA, monitor crowd levels, and generate tickets easily.

This project is inspired by real world transport apps and made for learning, college submission, and portfolio purposes.

>Features:

✔ Live Bus Tracking on Map
✔ Search Bus Routes
✔ ETA by Stop
✔ Crowd Level Monitoring
✔ Ticket Generator
✔ Nearest Stop Finder
✔ Real Time Clock
✔ Interactive Map using Canvas

>Technologies Used:

Python
Tkinter
Tkinter Canvas
OOP Concepts


>Project Structure:

main.py      # Starts the application
ui.py        # Main user interface
data.py      # Stores routes, stops, buses data
logic.py     # Contains movement and ticket logic


>How to Run

Install Python
Save all files in one folder
Run:
python main.py


>Modules Explanation

1. Live Map

Shows moving buses in real time on the map.

2. Route Search

Search buses by route number or route name.

3. ETA by Stop

Displays estimated arrival time for each stop.

4. Crowd Level

Shows passenger crowd level inside buses.

5. Ticket Generator

Generates ticket with route, fare, and timing.
